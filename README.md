**Expense Tracker Project**

A Python-based Expense Tracker System developed for Software Quality Engineering (SQE) assignment.
This project helps users manage daily expenses by adding, viewing, updating, and deleting expense records.

**Project Objective:**

The main objective of this project is to:

Develop a functional Python application
Apply manual and automated testing techniques
Perform bug reporting
Analyze code quality using SonarQube
Perform security analysis using Snyk
Use GitHub for version control

**Features**
Add new expenses
View all expenses
Update expense details
Delete expenses
Calculate total expenses
Search expenses by category/date

Python

unittest framework

**Installation**

Clone the repository:
git clone https://github.com/MahreenMalik05/ExpenseTrackerProject.git

Move into project directory:
cd ExpenseTrackerProject

Install dependencies:
pip install -r requirements.txt

Run the project:
python main.py

**Testing**

Run unit tests using:
python -m unittest
